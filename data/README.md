## Data Availability

This study uses data from the Alzheimer's Disease Neuroimaging Initiative (ADNI).

Due to ADNI data usage agreements:
- Raw data cannot be redistributed
- Users must obtain data directly from https://adni.loni.usc.edu/

This repository assumes the user has already obtained:
- ADNI-1
- ADNI-2
- ADNI-3

Only preprocessing scripts and feature definitions are provided here.
